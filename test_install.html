Once you've gone through the installation, you can test your installation to see if most things have been installed correctly.

On the Mac

To run the test script, you will need a program called Terminal. To open your Mac's Terminal app, you can search for Terminal in Spotlight, or you can find it in the /Applications/Utilities folder. Once it's open, you should see something like: 

Last login: Thu Jun 20 13:31:37 on ttys000
ComputerName:~ yourname$

Next, download this file by right clicking the link and choosing "Save link as". Save the file to your home folder, and make sure it is called workshop_checklist.py.

Now, type python workshop_checklist.py after the $ in the Terminal, and press enter. You should get an output something like this (but perhaps with newer version numbers):

Download this file by right clicking on it. Save the file to your home folder, and make sure it is called workshop_checklist.py.

Open your terminal by going to the Start menu. Select 'Run' and type cmd to run the cmd.exe Windows Command Prompt. On the Command prompt type: 

python workshop_checklist.py

and press Enter. You should get an output like this:

On Linux

Download this file. Save the file to your home folder, and make sure it is called workshop_checklist.py.

Open a terminal (making sure you're in the home folder) and at the command line run python workshop_checklist.py. If the printout says OK things are good. If it instead says FAILED, copy and send the entire output (all info above and summary below) to the instructor for help.
